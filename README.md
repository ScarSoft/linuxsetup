# Linux Setup Script

This repository contains a simple setup script (`setup.sh`) used to install
common system dependencies and run external installation utilities.

The script performs the following:

- Updates and upgrades the system
- Installs Python3 pip
- Runs external install scripts (example: Atuin)

---

## Download the Script

Clone the repository:

```bash
git clone https://github.com/ScarSoft/linuxsetup.git
cd linuxsetup
```
## Install the and run the script

```bash
chmod +x setup.sh
sudo ./setup.sh
```
